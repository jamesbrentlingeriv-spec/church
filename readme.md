## 🛠️ Admin Features

### Calendar Management

```
1. Click \"Add Event\"
2. Login: pastorron / godisgood
3. Add events persist in localStorage
4. Delete/edit events
5. Monthly navigation
```

### Logo Upload (admin.html)

```
1. Drag/drop PNG/JPG/GIF
2. Preview & remove
3. Persists via localStorage
4. Auto-scales in nav/footer
```
